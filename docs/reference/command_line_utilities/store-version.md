# version

Prints information about the TIBCO ComputeDB product version.

## Syntax

```pre
./bin/snappy version
```

## Example

```pre
./bin/snappy version
TIBCO ComputeDB Platform Version 1.1.0
    TIBCO ComputeDB RowStore 1.6.3 
    TIBCO ComputeDB Column Store 1.1.0 
    TIBCO ComputeDB AQP 1.0.3 
```
